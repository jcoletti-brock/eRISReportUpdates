# Hints for Data Analyst Role

## Writing and Maintaining Documentation

Any documentation that you generate and maintain should be structured under the `docs/datadesign` directory.

## Technology Selections

- **Database**: MSSQL is the recommended database engine.
- **Liquibase**: The sql migration tool is Liquibase its files are found in the services/backend/{service_name}/sql directory of each backend service. When making new SQL scripts, they should go here.
